# Phase 10 Scorekeeper

Μικρή web εφαρμογή στα ελληνικά για καταγραφή φάσεων και πόντων στο Phase 10.

## Λειτουργίες

- Προσθήκη/διαγραφή παικτών
- Κουτάκια φάσεων 1–10 για κάθε παίκτη
- Περιγραφή κάτω από κάθε φάση, ώστε να μην χρειάζεται να ανοίγεις συνέχεια τους κανόνες
- Η ενότητα **Φάσεις Phase 10** παραμένει διαθέσιμη και δεν έχει διαγραφεί
- Καταχώριση πόντων ανά γύρο για κάθε παίκτη ξεχωριστά
- Αυτόματο σύνολο πόντων δίπλα στο όνομα κάθε παίκτη
- Ιστορικό γύρων
- Αποθήκευση στο browser με localStorage
- Εξαγωγή JSON
- Light/Dark mode

## GitHub Pages

Ανέβασε τα αρχεία `index.html`, `styles.css`, `app.js` και `README.md` στο repository σου. Μετά πήγαινε:

`Settings` → `Pages` → `Deploy from a branch` → `main` → `/root`.
