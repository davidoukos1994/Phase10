# Phase 10 Scorekeeper

Μικρή στατική εφαρμογή για καταγραφή φάσεων και πόντων στο Phase 10.

## Λειτουργίες

- Προσθήκη και διαγραφή παικτών
- Κουτάκια φάσεων 1 έως 10 για κάθε παίκτη
- Καταχώρηση πόντων ανά γύρο
- Επιλογή αν ο παίκτης πέρασε τη φάση στον γύρο
- Ιστορικό γύρων ανά παίκτη
- Αυτόματος υπολογισμός συνολικών πόντων
- Ένδειξη πιθανού νικητή όταν κάποιος ολοκληρώσει και τις 10 φάσεις
- Αποθήκευση στο browser με localStorage
- Εξαγωγή δεδομένων σε JSON
- Dark mode

## Κανόνες σκορ

- Κάρτες 1–9: 5 πόντοι
- Κάρτες 10–12: 10 πόντοι
- Κάρτες μπλοκαρίσματος (!): 15 πόντοι
- Μπαλαντέρ (*): 25 πόντοι

## Ανέβασμα στο GitHub Pages

1. Δημιούργησε ένα νέο repository στο GitHub.
2. Ανέβασε τα αρχεία `index.html`, `styles.css`, `app.js`, `README.md`.
3. Πήγαινε Settings → Pages.
4. Στο Source διάλεξε `Deploy from a branch`.
5. Διάλεξε branch `main` και folder `/root`.
6. Πάτησε Save.

Μετά από λίγο η εφαρμογή θα είναι διαθέσιμη στο URL του GitHub Pages.
